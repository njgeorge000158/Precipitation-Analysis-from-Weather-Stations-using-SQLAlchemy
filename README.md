# Climate Analysis in Honolulu, Hawaii - Surfs Up!

In this project, I analyze precipitation data collected by weather stations in Honolulu, Hawaii, using SQLAlchemy Object Relational Model (ORM) queries, Pandas, and Matplotlib.  You can find the results of the analysis in the IPython notebook, climate_analysis.ipynb.  From the queries, I also created a RESTful Web API using Flask and Python, climate_api.py.
